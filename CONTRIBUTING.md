# Contributing

Issues are welcome.

External PRs may be deferred until v0.2 because the public API, script paths, and folder boundaries are still stabilizing.

Use the issue templates when reporting bugs or requesting features. Do not paste secrets, tokens, private keys, passphrases, private environment values, or private logs.

Terminal users can use `repo-automation/bin/repo-automation-report-upstream` to preview and optionally submit upstream bug or feature reports from the terminal.
